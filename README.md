#### install
  * tailwindcss
    https://nerdcave.com/tailwind-cheat-sheet
    
    ```bash
        $ npm i tailwindcss

        $ npm install -D tailwindcss@latest postcss@latest autoprefixer@latest

        $ npx tailwindcss init -p
    ```

### npm 
  * dev
    ```bash
      $ npm run build # デバッグモード
    ```
  * build
    ```bash
      $ npm run build # nextjsで作成されたファイルをhtml展開
    ```
     
  * start

    ```bash
      $ npm start # server 起動
    ```
